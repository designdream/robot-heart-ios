## Usage Guidelines

### Photography
- Always maintain aspect ratios when resizing
- Apply subtle dust/grain texture for playa authenticity
- Use warm color grading for sunset shots
- Preserve silhouettes in backlit images

### Logos
- Maintain minimum clear space (equal to robot's antenna height)
- Never distort or skew
- Ensure sufficient contrast with background
- Use cream logo on dark backgrounds only

### Composition
- Emphasize community and gathering
- Show scale through people in frame
- Capture golden hour lighting when possible
- Include geometric heart structure as focal point

## File Formats for Production

### Recommended Conversions
- Logos: Convert to PNG with transparency, SVG for scalability
- Photos: Optimize to WebP for web, keep original JPEG for print
- Icons: Export as PDF and PNG @1x, @2x, @3x for iOS

### iOS Asset Catalog
All assets should be added to `Assets.xcassets` with appropriate scale factors:
- @1x: Standard resolution
- @2x: Retina displays
- @3x: iPhone Plus/Pro Max displays
