# Robot Heart Color Palette

## Primary Colors

### Sunset/Golden Hour Palette
Extracted from iconic Robot Heart sunset imagery:

- **Burnt Orange** - `#D84315` / RGB(216, 67, 21)
  - Deep sunset orange, primary accent color
  - Used in gradients and highlights

- **Golden Yellow** - `#FFB300` / RGB(255, 179, 0)
  - Warm golden hour glow
  - Secondary accent, warmth

- **Deep Red-Orange** - `#BF360C` / RGB(191, 54, 12)
  - Rich sunset red
  - Dramatic accent

- **Amber** - `#FF6F00` / RGB(255, 111, 0)
  - Mid-tone sunset
  - Transitional color

### Robot Character Colors

- **Robot Cream** - `#E8DCC8` / RGB(232, 220, 200)
  - Primary robot body color
  - Light UI elements, text on dark

- **Robot Brown** - `#3D2817` / RGB(61, 40, 23)
  - Robot features (eyes, mouth, heart dots)
  - Dark details

### Background Colors

- **Playa Dust** - `#C4A57B` / RGB(196, 165, 123)
  - Desert sand tone
  - Neutral background

- **Deep Night** - `#1A1410` / RGB(26, 20, 16)
  - Near-black background
  - Primary dark mode background

- **Warm Gray** - `#2A1F1A` / RGB(42, 31, 26)
  - Secondary dark background
  - Card backgrounds

### Accent Colors

- **Turquoise Sky** - `#4ECDC4` / RGB(78, 205, 196)
  - Bright playa sky
  - Active states, location markers

- **Dusty Pink** - `#FF8B94` / RGB(255, 139, 148)
  - Soft accent
  - Secondary highlights

- **LED Magenta** - `#E91E63` / RGB(233, 30, 99)
  - Stage lighting
  - Alert/attention color

## Gradient Combinations


### Playa Gradient
```
Top: #4ECDC4 (Turquoise Sky)
Bottom: #C4A57B (Playa Dust)
```

### Dark Mode Gradient
```
Top: #2A1F1A (Warm Gray)
Bottom: #1A1410 (Deep Night)
```

## Usage Guidelines

### Text
- **Primary Text (Dark Mode)**: Robot Cream `#E8DCC8`
- **Secondary Text**: 70% opacity Robot Cream
- **Disabled Text**: 40% opacity Robot Cream

### Backgrounds
- **Primary**: Deep Night `#1A1410`
- **Secondary**: Warm Gray `#2A1F1A`
- **Elevated**: 10% lighter than Warm Gray

### Accents & CTAs
- **Primary CTA**: Burnt Orange `#D84315`
- **Secondary CTA**: Golden Yellow `#FFB300`
- **Active State**: Turquoise Sky `#4ECDC4`

### Status Colors
- **Connected**: `#4CAF50` (Green)
- **Warning**: `#FF9800` (Orange)
- **Error**: `#F44336` (Red)
- **Info**: Turquoise Sky `#4ECDC4`

## Color Psychology

**Warm Sunset Tones**: Evoke the magical golden hour on the playa, community gathering, warmth, and connection.

**Robot Cream**: Approachable, friendly, nostalgic - makes the robot character feel warm despite being mechanical.

**Turquoise Sky**: Fresh, energetic, optimistic - represents the vast playa sky and freedom.

**Deep Darks**: Practical for bright desert sun, reduces eye strain, creates intimate atmosphere for night events.
